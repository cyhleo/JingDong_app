# JingDong_app
利用抓包工具mitmdump对接python，爬取京东商品评论。

# 该项目的使用方法：
1. 执行mitmdump -p 端口号 -s mitm_jd.py。
2. 将计算机的主机名和端口号填入手机的代理设置中。
3. 使用手机浏览京东商品的评论信息，即可实现抓取评论数据，并将数据持久化到MongoDB数据库中。

# 项目说明
该项目结合airtest自动化测试工具可实现，商品评论的自动抓取。
